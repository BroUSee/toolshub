Color Values

HEX vs RGB — What is the difference?

HEX colors use a 6-digit hexadecimal code like #FF5733. RGB uses three numbers for Red, Green and Blue values (0-255 each) like rgb(255, 87, 51). Both represent the same colors — CSS supports both formats. RGBA adds a fourth value for opacity/transparency.
